# 图文大模型与 AR 沉浸式交互系统

国家级大学生创新训练项目的公开技术概览。项目围绕展馆讲解与沉浸式交互，设计“图像理解 - 文本问答 - 可控图像生成 - AR 展示”的多模态应用链路。

[在线项目页](https://yunyao-9527.github.io/multimodal-ar-exhibition-system/)

> 本仓库用于公开展示系统设计、技术路线与个人项目贡献。原始申报材料、票据、用户数据和未授权资源不在公开范围内。

## 技术路线

1. 使用 InternVL2 对展品或场景图像进行理解，生成结构化描述与问答上下文。
2. 使用本地大模型与 DeepSeek 完成知识问答、讲解内容组织和交互意图解析。
3. 通过 Stable Diffusion 与 ComfyUI 工作流生成指定风格的视觉内容。
4. 在 Linux 环境通过 Ollama 部署并量化本地模型，以 RESTful API 连接 AR 前端。

## 个人贡献

- 负责图文大模型与生成式模型的技术调研、方案设计和原型链路实现。
- 完成 InternVL2、Stable Diffusion、ComfyUI、Ollama 与 DeepSeek 等组件的选型与集成设计。
- 设计模型服务 API 与 AR 前端之间的数据交互流程，并参与项目答辩材料与结项报告整理。
